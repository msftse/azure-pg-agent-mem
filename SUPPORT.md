# Support

## How to file issues and get help

This project uses [GitHub Issues](https://github.com/msftse/azure-pg-agent-mem/issues) to track bugs and feature requests. Please search the existing issues before filing new issues to avoid duplicates. For new issues, file your bug or feature request as a new issue.

For help and questions about using this project, please use the following resources:

- Review the [README](README.md) for setup and configuration guidance
- Set `AGENT_MEM_LOG_LEVEL=DEBUG` for verbose diagnostic output
- Check `npx tsx src/index.ts db status` to verify database connectivity

## Microsoft Support Policy

Support for this project is limited to the resources listed above.
